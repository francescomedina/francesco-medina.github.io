Publications

Analysis and Contributions to an Open Source Kyber Library in Rust

Authors: F. Medina, M.C. Molteni, A. Di Scala, L. Nava

Presented in: 1st Workshop on Quantum-Secure Networks and Systems (QSNS2024), co-located with the 29th IEEE Symposium on Computers and Communications

Date: June 26-29, 2024

Location: ESIEE PARIS, France

This paper focuses on analyzing the efficiency of a purely Rust-written Kyber library for ARM Cortex M4 microcontrollers. The analysis includes performance comparisons with a C-written Kyber library, as well as the identification and resolution of issues within the Rust-written Kyber library. Contributions include solutions for unhandled True Random Number Generator (TRNG) exceptions, enhancements in code quality, and improvements in code coverage, all of which have been approved by the library maintainers and released.
